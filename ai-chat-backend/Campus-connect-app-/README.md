# Campus Connect App

A comprehensive mobile application for university students, faculty, and staff to access academic information, campus services, events, and more.

## Features

### Core Features
- **Dashboard** - Quick access to all features with personalized greeting
- **Course Management** - View enrolled courses, register for new courses
- **Grades & Performance** - Track academic performance and CGPA
- **Event Calendar** - Browse and register for campus events
- **Campus Map** - Interactive map with building navigation
- **Profile Management** - View and edit personal information

### Services
- **Fees & Payments** - View fee details and make payments
- **Transport Tracking** - Live bus tracking and schedules
- **Hostel Services** - Mess menu, maintenance requests
- **Health Center** - Book appointments, view health records

### Safety
- **Emergency Services** - Quick SOS button, emergency contacts
- **Report Incidents** - Report safety concerns

## Tech Stack

- **Frontend**: Flutter (Dart)
- **State Management**: Provider
- **Backend**: Firebase (optional)
- **Maps**: Google Maps SDK (optional)
